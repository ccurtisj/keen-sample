# keen-sample
Quick sample of Keen API integration. Covers server-side publishing, front-end publishing, and basic reporting.

## Interesting files

* `keen-sample.rb` // server-side
* `public/application.js` // client-side + reporting


## Start sinatra

```
$ bundle
$ rackup
```
